# Agronomía Pastura

Proyecto desarrollado para la materia Programación III - Frontend.

## Descripción

Agronomía Pastura es un sitio web informativo orientado a la comercialización de pasturas, gramíneas y leguminosas. El proyecto permite visualizar información sobre los productos ofrecidos, consultar datos climáticos en tiempo real y contactar a la empresa mediante un formulario validado con JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- API Open-Meteo
- Git y GitHub

## Funcionalidades

- Sitio web multipágina.
- Diseño responsive para dispositivos móviles, tablets y escritorio.
- Menú de navegación responsive.
- Formulario de contacto con validaciones.
- Consumo de API meteorológica mediante Fetch.
- Visualización de clima actual.
- Visualización de clima extendido.
- Búsqueda dinámica de datos climáticos.
- Uso de Bootstrap para maquetación y componentes.

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/Agostinarollan/Proyecto-Front-Programacion3.git
```

2. Abrir el proyecto en Visual Studio Code.

3. Ejecutar con Live Server o abrir `index.html` en el navegador.
